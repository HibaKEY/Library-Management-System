# Library Management System

**Project:** Library Management System  
**Date:** Mar 14, 2025  
**Technologies:** Python + Tkinter  
**Role:** Developer

## Description
Automated management of a library with 50+ books including loan tracking and returns.  
The system streamlines book management and improves operational efficiency.

## Key Outcomes
- Managed 50+ books with automated loan tracking  
- Reduced manual errors and improved tracking accuracy  
- Resulted in **30% improvement in operational efficiency**

## Features
- Add / remove books  
- Track borrowed and returned books  
- Display available books  
- Simple GUI built with Tkinter
